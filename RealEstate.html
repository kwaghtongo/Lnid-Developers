The Objective of this project is to find a home acceptable to all family at right place at good cost.There are seven modules in this project. At first the User should Register himself in order to access the Application. Login Module Allows only the Authenticated Users.


Market Data Module is used to view historic, current and projected trends of rates at the sub market level. Descriptive Data Module describes about the available plots Information such as Name, Address and Structure. User should pay some Initial amount to reserve a plot by Online Booking. Transaction module describes the properties last sold, the Buyer and the selling price etc.


Project Description Modules required are:

  • Registration
  • Login
  • Market Data Module
  • Descriptive Data Module
  • Plots Booking Module
  • Transaction Data Module
  • Plots selling Module
